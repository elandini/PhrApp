1. Installazione ed esecuzione

Disinstallare qualunque distribuzione di Python già presente di Python (se non si ha dimestichezza con l'installazione dei pacchetti). Scaricare quindi Python XY (python XY) e seguire le instruzioni fornite dal sito degli sviluppatori.

Attenzione: Solo per utenti Windows

Fatto ciò aprire un prompt dei comandi, su windows ("tasto windows"+r, digitare "cmd" e premere Invio), o un terminale, su Linux, e digitare "easy_install nome_pacchetto", dove a nome_pacchetto vanno sostituiti, nell'ordine, i seguenti nomi:

  1. sendkeys

  2. ctypes

A questo punto, bisogna scaricare PyWinAuto (Scarica PyWinAuto). Estrarre quindi la cartella nel file .zip, aprire un prompt dei comandi, entrare nella cartella appena estratta e digitare "python setup.py install". Terminato il procedimento di installazione, aprire la cartella di QPM-python e fare doppio click su "QPM-GUI.py"

Torna al sommario


2. Apertura file


Gruppi di file .tif

Cliccare sul menù "File" o premere Atl+f e scegliere "Open". Cercare la cartella con le immagini e selezionare quelle da utilizzare.


File nd2

Attenzione: Solo per utenti Windows
Se avete installato "NIS Element Viewer" di Nikon (Download), Aprire il menù "File" e selezionare "Open Nd2". Selezionare, quindi, il file e, successivamente, la cartella in cui salvare i file tif esportati dal file .nd2. Una volta scelta la cartella, il programma eseguirà in maniera automatica "NIS Element Viewer" ed esporterà i .tif nella cartella selezionata. Fino a che l'operazione non sarà terminata (ovvero quando verrà richiuso il software Nikon) si prega di non interagire con il pc. Questo perchè andreste a disturbare il funzionamento dello script che controlla il programma Nikon.

3. Impostazioni iniziali

Una volta caricate le immagini, si potrà impostare l'immagine con il fuoco migliore. Per farlo, si possono scorrere le immagini usando la barra di scorrimento sotto al visualizzatore o usando il menù a tendina che ne riporta i path e decidere autonomamente quale sia la migliore, cliccando sulla Checkbox posta di fianco alla barra di scorrimento. Altrimenti, si può affidare al software il compito di sceglierla, cliccando sul radio button "Autofocus". Cliccando nuovamente sul checkbox relativo ad una immagine diversa da quella scelta dal software, sposterà la selezione del radio button su "Manual". Si può inoltre impostare il parametro "Alpha corrector" usato nel calcolo di ricostruzione dell'immagine (Spiegazione omissis per il momento). I valori consigliati rimangono nel range 10-6 a 10-2. Va poi impostato il passo tra le immagini in nanometri.

4. Creazione e salvataggio immagini 3D

Terminate le impostazioni iniziali, si può passare alla creazione dell'immagine 3D. Cliccando su "Create 3D Image" partirà l'algoritmo di ricostruzione. Una volta terminato, sarà possibile visualizzare il risultato cliccando su "View 3D Image". Verranno quindi aperte due finestre, una con il risultato visto in due dimensioni e un render in 3D navigabile del risultato. Nel caso il risultato sia quello atteso, si potrà procedere a salvare l'immagine cliccando su "Save 3D Image". Prima di salvare vanno impostati il nome del file e l'estensione (selezionabile dall'apposito menù a tendina) e la cartella, che di default è quella che da cui sono state caricate le immagini (per cambiare directory, basta cliccare sul pulsante "...").
